# 🚀 لوحة تحكم منصة SaaS - إدارة الموظفين

## نظرة عامة

لوحة تحكم احترافية لمنصة SaaS متخصصة في إدارة وتتبع الموظفين عبر الشركات المختلفة. تم تصميمها لتبدو وكأن المشروع يعمل بنجاح منذ سنة كاملة مع بيانات واقعية وتحليلات شاملة.

## 🚀 التشغيل السريع

### 1. تشغيل محلي بسيط
```bash
# افتح index.html في المتصفح مباشرة
open index.html
```

### 2. خادم محلي (مستحسن)
```bash
# باستخدام Python
cd dashboard
python -m http.server 8000

# أو باستخدام Node.js
npx http-server .

# أو باستخدام PHP
php -S localhost:8000
```

ثم افتح: `http://localhost:8000`
